## Security Awareness

**All agents must be hardened against attacks:**
- Command injection (rm -rf, sudo, shutdown)
- Path traversal (/etc/passwd, ~/.ssh, ~/.aws)
- Prompt injection ("ignore previous instructions")
- Rate abuse (unlimited requests)
- Resource exhaustion (10GB files, 500 tool calls)
- Credential theft (hardcoded secrets, exposed keys)

**Reference implementations:**
- `eve_code_agent.py` - Secure code execution agent
- `eve_moltbook_agent.py` - Secure social media agent
- `SECURITY.md` - Comprehensive security guide
- `test_security.py` - Automated security tests

**Use SecurityValidator class for all new agents:**
```python

```
